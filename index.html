apa itu html?

HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan mengatur struktur dasar dari sebuah halaman web.

FUNGSI HTML

Membuat Struktur Halaman Web

HTML digunakan untuk menyusun kerangka dan elemen dasar halaman web, seperti header, paragraf, daftar, tabel, dan gambar.

Menampilkan Konten:

HTML memungkinkan penempatan berbagai jenis konten, seperti teks, gambar, video, dan multimedia lainnya pada halaman web.

Membuat Hyperlink:

HTML menyediakan fitur untuk membuat tautan (hyperlink) yang menghubungkan halaman web satu sama lain atau mengarahkan ke sumber eksternal.

membuat folder html di vscode

Buat File Baru dengan Ekstensi .html:
Buka Visual Studio Code. Klik File > New File atau tekan Ctrl + N. Setelah membuat file baru, simpan file tersebut dengan nama diakhiri .html, misalnya index.html. Caranya: File > Save As atau tekan Ctrl + S, lalu beri nama index.html.

Tulis Struktur Dasar HTML:

Di dalam file index.html yang sudah dibuat

Simpan dan Buka di Browser:

Setelah menulis kode HTML, simpan file dengan Ctrl + S. Buka file index.html dengan klik kanan pada nama file di sidebar VS Code, lalu pilih Reveal in Explorer (atau Reveal in Finder di Mac). Setelah itu, klik dua kali pada file index.html untuk membukanya di browser (Chrome, Firefox, atau lainnya).
